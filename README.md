# Keybinds

> **sin** -> **Shift + S**

>  **cos** -> **Shift + C**

>  **tan** -> **Shift + T**

>  **square root** -> **S**

>  **cubic root** -> **T**

>  **reciprocal** -> **R**

>  **square** -> **K**

>  **plus/minus** -> **N**

>  **erase** -> **BackSpace**

>  **clear** -> **C**

>  **clear entity** -> **Shift + E**

>  **equal** ->  **Enter**
